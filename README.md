parallel-computing
==================

Parallel computing project for school

## Requirments 

* a compatible terminal emulator installed:
** urxvt
** konsole
** xterm
* g++
* gdb
* mpi library installed
* boost installed

## Compile

### Normal

```bash
$ make
```

### Debug

```bash
$ make debug
```

## Run

### Normal

```bash
$ make run
```

### Debug

```bash
$ make rundebug
```
